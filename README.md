# Template - Parcel

## Description

Template d'un projet

## Initialisation

```
$ npm init -y
```

## Commandes

**Dev:**
lancer start_dev.bat
ou
```
$ npm run dev
```
ou
```
$ ./node_modules/.bin/parcel index.html
```

**Build:**
lancer start_build.bat
ou
```
$ npm run build
```
ou
```
$ ./node_modules/.bin/parcel build src/pages/index.html -d build
```
