# Template - Parcel

## Description

Template d'un projet

## Initialisation

```
$ npm install
```

## Commandes

**Dev:**

lancer start_dev.bat | ```$ npm run dev``` | ```$ ./node_modules/.bin/parcel index.html```

**Build:**

lancer start_build.bat | ```$ npm run build``` | ```$ ./node_modules/.bin/parcel build src/pages/index.html -d build```
